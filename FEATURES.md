# P5136 support status

## 확인됨

- P5136 실행 파일과 PIN 기반 빌드 감지
- P5136 포트 토폴로지와 접속기 XML/PIN 설정
- 로그인과 메인 메뉴 진입
- 기본 인벤토리, 마이룸과 메인 메뉴 전환
- 방 목록, 방 생성과 방 설정 변경
- 구체적인 맵 선택과 경기 시작 패킷 반영
- 서버와 접속기 분리 실행
- TCP/UDP 송수신 trace 기능과 로컬 로그 저장
- UDP P2P endpoint 교환 및 서버 relay 기반 코드 경로

## 부분 구현 또는 미검증

- 2인 이상 방 참가와 실주행 동기화
- 팀전, 아이템전과 모든 게임 모드 조합
- 관전자 동기화
- 클라이언트가 보낸 랜덤 맵 후보 배열을 사용한 최종 서버 선택
- 경기 종료와 모든 결과·보상 조합
- WAN/NAT 환경의 P2P fallback
- 패킷 재정렬, jitter buffer 또는 서버 측 위치 보간

## 알려진 제한

- 서버는 P5136 게임 데이터가 있는 폴더에서 빌드를 감지하므로 독립된 빈 폴더에서는 시작하지 않습니다.
- 공식 릴리스는 게임 클라이언트나 데이터를 포함하지 않습니다.
- 실행 파일은 코드 서명되지 않았습니다.
- 전체 패킷 trace는 사용자명, 주소와 인증 데이터를 포함할 수 있습니다.
- 원격 카트 상태는 클라이언트 주도 P2P와 서버 fallback relay 구조이며 서버 authoritative 물리 시뮬레이션이 아닙니다.
